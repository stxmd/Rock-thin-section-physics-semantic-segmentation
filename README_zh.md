# 1. 项目概述

![alt text](gifs/Rock_thin_section_preview_zh.gif)

在正交偏光显微镜下，非等轴晶系矿物的光学表现与其相对于起偏器和检偏器的取向密切相关。随着显微镜载物台旋转，同一岩石薄片中已配准的正交偏光显微图像在不同旋转状态下仍保持稳定的空间对应关系，即矿物颗粒的位置、边界和形态基本不变；与此同时，其透射光强、干涉色和消光状态会随旋转角度发生周期性变化。

这种“空间语义稳定、光学表现变化”的物理特性为正交偏光岩石薄片图像的语义分割提供了重要启发。基于这一特性，本研究推导并设计了一种面向正交偏光岩石薄片图像的数据增强方法，用于在保持矿物语义类别、空间边界和形态结构一致的前提下，引入符合偏光显微成像规律的光学变化。

本仓库为相关论文的预告仓库。待论文审稿流程完成后，仓库将逐步整理并公开该方法的完整实现流程、实验设置、数学论证、可视化材料及相关研究资源，以支持学术交流与结果复现。
